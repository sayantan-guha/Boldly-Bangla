# Boldly Bangla — Hoichoi Claude Plugin

Bengali copywriting skill for Hoichoi's Claude (Cowork) users.

## Install

1. Open Claude desktop → Plugins
2. Click **Install from URL** (or drag-drop the `.plugin` file)
3. Use the latest release URL:  
   `https://github.com/hoichoi/<REPO_NAME>/releases/latest/download/boldly-bangla.plugin`

## What it does

Gives Claude deep knowledge of Bengali tone, idiom, cultural references, and Hoichoi brand voice — for writing ad copy, social captions, campaign taglines, and OTT content in Bangla.

## Updating

Sayantan maintains this repo. When skill files are updated and pushed, pull the latest `.plugin` file from the Releases page and reinstall.

## Folder structure

```
SKILL.md                   ← Main skill prompt
copywriting-principles.md  ← Core writing rules
personas/                  ← Voice personas (Chandril, Anindya, Rajatava…)
cultural-references/       ← Source material by person/event
references/                ← Style exemplars, cultural calendar, grammar
literature/                ← Bengali literary references
cinema/                    ← Film & media references
sports/                    ← Sports culture
language/                  ← Grammar & Guruchandali
food-society/              ← Food & social culture
history/                   ← Historical references
```
